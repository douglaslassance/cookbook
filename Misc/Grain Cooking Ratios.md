[]()The technique for grains is generally the same.

1. Rinse the grains.
2. Pour into the pot along with the corresponding water ratio.
3. Bring to a boil.
4. Cover and simmer until all water has evaporated.

| Type                   | Water Ratio |
| ---------------------- | ----------- |
| Sushi Rice             | 100%        |
| Jasmin Rice            | 100-125%    |
| Basmati Rice           | 150%        |
| Long Grain Brown Rice  | 125%        |
| Short Grain Brown Rice | 175%        |
| Forbidden Rice         | 175%        |
| Wild Rice              | 300%        |
| Steel-cut oats         | 300-400%    |
| Keisha                 | 175%        |
| Pearl Barley           | 300%        |
| Hulled Barley          | 300%        |
| Semolina               | 100%        |
| Bulgur                 | 150-200%    |
| Israeli Couscous       | 150%        |
| Quinoa                 | 200%        |
| Farro                  | 300-400%    |
| Rolled oast            | 200%        |
